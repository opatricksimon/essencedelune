# Essence de Lune

Projeto inicial da landing page da marca `Essence de Lune`.

## Objetivo do primeiro release

Criar uma landing page simples para trafego vindo do Instagram com um unico objetivo:

- capturar lead
- entregar um guia gratuito
- registrar os dados em Google Sheets

## Arquitetura recomendada

Para este caso, a versao mais simples e mais segura para sair rapido e:

1. Landing page estatica em HTML, CSS e JS.
2. Formulario customizado embutido na pagina.
3. Envio do formulario para Google Sheets via Google Apps Script Web App.
4. Tela de sucesso com link para baixar o guia ou abrir o guia.

## O que fica para depois

- automacao de email
- CRM
- multiplas paginas
- blog
- loja
- animacoes mais pesadas
- stack complexa
